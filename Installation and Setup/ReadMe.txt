
Installation and Setup

1) Please go the project repository on Github. 
2) Find the “Installation and Setup” directory.  
3) In this directory, find the “app-debug.apk” file. 
4) Download this file on your computer and send it to your phone through USB (in some directory where you can locate it.) 
5) Then, on your phone, use the File Manager to locate the file, and tap on it to install the app.
